┌─────────────────────────────────────────────────────────────────────────────────┐
│                              SECURITY FEATURES                                │
└─────────────────────────────────────────────────────────────────────────────────┘

🔐 PASSWORD SECURITY:
   • Passwords encrypted using Affine Cipher (key=23)
   • Educational demonstration of classical cryptography
   • Easily reversible for password recovery

🛡️ FILE SECURITY:
   • File contents encrypted using RSA (1024 or 2048-bit keys)
   • Public/Private key pairs stored securely
   • Digital signatures for integrity verification
   • Binary storage of encrypted content

👥 USER ISOLATION:
   • Files isolated by user email
   • Users can only access their own data
   • Database-level access control
